# Gemini Follow-up Prompt — Turn the Research into Teachable Assets

Take the report you just produced and convert it into classroom-ready assets for a five-day, one-hour-per-day course.

## What I need now

Transform the prior research into:
1. a final course outline
2. a final slide outline
3. an exercise pack
4. a Claude Code CLI lab guide
5. a repo scorecard handout
6. a capstone brief

## Constraints

- Claude Code CLI remains the central hands-on tool.
- Azure is the default platform context.
- .NET is the primary example path.
- Python is the secondary example path.
- Brownfield improvement remains the main use case.
- Keep the content practical and instructor-ready.
- Do not start over with new theory. Convert the prior report into usable teaching materials.

## Output structure

Return Markdown with these sections:

# Final Course Outline
# Day-by-Day Slide Outline
# Claude Code CLI Lab Guide
# Exercises and Handouts
# Repo Health Scorecard
# Capstone Brief
# Instructor Notes
# Optional Content to Cut if Time Is Tight

## Style

- crisp
- practical
- no filler
- no repetition unless it is useful for teaching
- concrete examples over generic claims
