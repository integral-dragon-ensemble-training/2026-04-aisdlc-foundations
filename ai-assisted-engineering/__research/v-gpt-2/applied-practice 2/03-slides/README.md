# Slide Folder Guide

This folder contains detailed markdown for slide creation.

## Files

- `00-course-overview-slide.md` — one overview slide that frames the entire approach
- `01-day-1-what-good-looks-like.md`
- `02-day-2-technical-debt-as-friction.md`
- `03-day-3-using-ai-to-inspect-and-improve.md`
- `04-day-4-validation-tests-observability-security.md`
- `05-day-5-making-it-repeatable.md`

## Suggested usage

- Start each day with the corresponding deck.
- Reuse the overview slide on Day 1 and at the start of any executive or leadership-facing version of the class.
- Trim each day to 5–7 slides depending on discussion depth.

## Recommended slide-building pattern

For each slide, convert:
- Core message → slide headline
- Bullets → body content
- Suggested visual → diagram or screenshot direction
- Speaker notes → presenter notes
