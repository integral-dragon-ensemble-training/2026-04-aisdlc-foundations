# Workshop 3 — Using AI to Inspect and Improve

Module 02 · Workshop 03 of Integral Dragon's AI-SDLC Applied Practice week.

## Summary

Workshop 3 moves the class from "AI is impressive" to "AI is useful when aimed carefully." Participants learn the five reliable jobs AI does well on existing codebases (inspect, explain, propose, scaffold, standardize), practice tightening prompts, study AI failure modes, and produce inspectable artifacts against a real repository.

Framing lines for the workshop:

- Use AI to make the hidden visible.
- Trust evidence, not fluency.
- Small, scoped, reviewable — always.
