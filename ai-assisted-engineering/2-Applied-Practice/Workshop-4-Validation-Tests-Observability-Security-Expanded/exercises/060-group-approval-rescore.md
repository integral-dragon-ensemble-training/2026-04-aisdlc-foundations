# 060 Group Approval Rescore

## Objective

Prepare the branch for group approval and rescore affected project-health categories.

## Claude Prompt

```text
Prepare my Workshop 4 group approval packet.

Include:
- branch name
- change summary
- validation matrix
- tests run
- observability evidence
- security evidence
- approve/revise/reject recommendation
- affected scorecard categories
- proposed rescore and evidence
- what remains unproven

Write the packet to 060-group-approval-rescore/group-approval-packet.md.
```

## Output

- group approval packet
- rescore proposal
