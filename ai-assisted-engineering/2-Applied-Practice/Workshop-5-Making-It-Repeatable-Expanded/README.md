# Workshop 5 Expanded - Making It Repeatable

This expanded version turns Workshop 5 into the bridge from individual AI-assisted practice to team operating model.

## Thesis

The value is not one good AI-assisted cleanup. The value is a repeatable way for the team to find friction, improve safely, review together, and adopt changes deliberately.
