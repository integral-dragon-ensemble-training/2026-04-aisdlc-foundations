# 030 Team Working Agreements For AI

## Objective

Define how the team will use AI-assisted engineering safely and consistently.

## Claude Prompt

```text
Draft team working agreements for AI-assisted development.

Cover:
- when AI is useful
- when AI should not be used alone
- branch naming
- coursework or evidence artifacts
- review expectations
- validation expectations
- security and secrets handling
- how to document AI-assisted changes

Make the agreements short enough for a team to actually use.
```

## Output

- team working agreements
- open policy questions
