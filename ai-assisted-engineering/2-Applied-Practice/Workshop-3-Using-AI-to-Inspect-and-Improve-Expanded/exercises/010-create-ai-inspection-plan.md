# 010 Create AI Inspection Plan

## Objective

Convert one Workshop 2 friction item into a bounded AI inspection plan.

## Claude Prompt

```text
We are starting Workshop 3: Using AI to Inspect and Improve.

Use my Workshop 2 friction register and group review packet if available.

Create an AI inspection plan for one selected friction item.

The plan must include:
- target question
- files or areas likely relevant
- what evidence to collect
- what not to change yet
- risks of overreach
- expected output format

Do not inspect broadly yet. Show me the plan first.
```

## Output

- inspection plan
- selected friction item
- scope boundary
