# 030 Prompt Tightening And Evidence

## Objective

Practice turning broad AI output into evidence-backed engineering output.

## Claude Prompt

```text
Review your previous answer for weak evidence.

Find:
- claims without file/path evidence
- inferred architecture stated too strongly
- suggested changes that exceed scope
- missing tests or validation notes
- uncertainty that should be explicit

Rewrite the findings so every important claim has evidence or an uncertainty note.
```

## Output

- tightened findings
- evidence gaps
- improved prompt notes
