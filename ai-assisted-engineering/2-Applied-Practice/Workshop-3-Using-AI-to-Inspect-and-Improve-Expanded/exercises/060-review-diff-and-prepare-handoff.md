# 060 Review Diff And Prepare Handoff

## Objective

Prepare the candidate change for human review and Workshop 4 validation.

## Claude Prompt

```text
Review the candidate diff.

Produce a handoff packet with:
- branch name
- inspection question
- evidence collected
- files changed
- validation run
- what passed
- what failed
- what remains unproven
- risks for Workshop 4 validation
- recommendation: merge, revise, or discard

Write this to 060-review-diff-and-prepare-handoff/review-packet.md.
```

## Output

- review packet
- Workshop 4 validation inputs
