# Applied Practice — Course Overview

Week 3 of the AI-Accelerated Software Development course for Ensemble Health Partners. This folder is the single-page orientation for instructors and students: the thesis, the framework, the debt taxonomy, the five-day map, and the overview deck.

## Files

### Foundation

- [`course-thesis-and-storyline.md`](./course-thesis-and-storyline.md) — the main claim, the story arc, and phrases worth repeating in class.
- [`what-good-looks-like-framework.md`](./what-good-looks-like-framework.md) — nine categories of project health, with debt signals and AI guidance.
- [`technical-debt-taxonomy.md`](./technical-debt-taxonomy.md) — nine debt categories, how to explain interest, where AI has leverage.

### Course design

- [`five-day-course-map.md`](./five-day-course-map.md) — the five sessions, learning objectives, and lesson blocks.
- [`daily-lesson-outlines.md`](./daily-lesson-outlines.md) — opening moves, key points, activities, and exit prompts per day.
- [`topic-bank-and-cut-list.md`](./topic-bank-and-cut-list.md) — must-teach, nice-to-have, and cut-first topics for pacing decisions.

### Overview deck

- [`course-overview-slides.html`](./course-overview-slides.html) — nine-slide Integral Dragon deck: title, thesis, five-part method, five sessions, nine health categories, nine debt categories, AI helps vs. risky, 30-day plan, close.
- [`course-overview-slides.pdf`](./course-overview-slides.pdf) — rendered PDF (1280×720, one page per slide).

## Where this fits

This `overview/` folder is the entry point. Each session has its own folder (`Session-1-What-Good-Looks-Like/` through `Session-5-Making-It-Repeatable/`) containing slides, exercises, resources, and instructor notes.
