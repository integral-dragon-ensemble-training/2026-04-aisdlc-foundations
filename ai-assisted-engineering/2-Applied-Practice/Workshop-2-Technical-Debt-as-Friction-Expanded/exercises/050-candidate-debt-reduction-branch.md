# 050 Candidate Debt Reduction Branch

## Objective

Apply or draft one small debt-reduction improvement on the participant branch.

## Claude Prompt

```text
Prepare one small debt-reduction change for the selected friction item.

Constraints:
- use branch workshop-2-technical-debt-as-friction/<first-last-kebab>
- keep the diff small
- prefer docs, setup clarity, test confidence, or workflow clarity over broad refactoring
- show the plan before editing
- run the narrowest relevant validation
- copy summary, diff, validation result, and remaining uncertainty to the coursework folder
- do not merge or push unless I explicitly approve
```

## Output

- selected improvement plan
- branch summary
- validation result
- rescore or friction-reduction note
