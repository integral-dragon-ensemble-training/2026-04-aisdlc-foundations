## Core Positioning

The executive story is:

```text
AI coding tools create value fastest when they are attached to a repeatable engineering improvement loop.
```

The Applied Practice workshops turn unmanaged AI coding usage into a governed workflow:

```text
real repo
  -> AI-assisted inspection
  -> human-reviewed insights
  -> participant branch experiments
  -> validation gate
  -> group review
  -> selected improvements
  -> repeatable operating model
```

## Recommended Use

- Use `010-executive-one-page-brief.md` as the first email or meeting handout.
- Use `020-benefits-and-rapid-delivery.md` to explain why the value appears quickly.
- Use `030-insight-examples.md` as a source for slide examples and executive readouts.
- Use `image-prompt-guides/` to generate visuals for decks, emails, or leadership readouts.

## Language Guidance

Use "insights" instead of "metrics" when selling the approach. The point is not to create a political scorekeeping system. The point is to make engineering friction visible enough that teams can act on it.
