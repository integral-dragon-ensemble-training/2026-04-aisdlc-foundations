# GPT Image Model Notes

As of April 22, 2026, OpenAI's developer documentation lists `gpt-image-2` as the current state-of-the-art image generation model for fast, high-quality image generation and editing.

## Practical Implications For This Marketing Work

- Use `gpt-image-2` for new executive visuals when available.
- Use `gpt-image-1.5` only if `gpt-image-2` is unavailable in the target account or tool surface.
- Ask for 16:9 presentation-ready images when the visual is intended for slides.
- Keep text sparse. Even improved image models can create presentation visuals more reliably when labels are short and limited.
- Use iterative editing for refinement: generate the overall composition first, then edit one aspect at a time.
- Provide source diagrams or screenshots as references only when brand fidelity or layout fidelity matters.

## Prompting Guidance For Executive Visuals

Strong prompts should specify:

- Audience: executive leadership, technology leadership, engineering management.
- Use case: board deck, executive readout, leadership email, strategy presentation.
- Format: 16:9 slide visual, clean infographic, white or light neutral background unless a dark theme is specifically desired.
- Visual metaphor: operating loop, transformation pipeline, before/after dashboard, team review workflow.
- Text limits: no dense text, no paragraphs, maximum 5-7 labels.
- Brand safety: no fake logos, no real client names, no sensitive system names.
- Output quality: polished consulting-style visual, crisp shapes, high contrast, readable label areas.
