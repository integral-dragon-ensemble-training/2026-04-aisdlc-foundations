# Prompt Guide: Executive Operating Model Visual

## Use Case

Use this for a leadership slide that explains the Applied Practice approach at a glance.

## Base Prompt

```text
Draw a 16:9 executive presentation visual for technology executives showing how AI coding becomes a governed engineering operating model.

Style: polished consulting-style infographic, crisp vector shapes, light neutral background, deep navy and muted green accents, high contrast, modern but not flashy.

Show a left-to-right operating loop with five stages:
1. Real Repo
2. AI Inspection
3. Human Review
4. Safe Branch
5. Validated Adoption

Below the loop, show a thin evidence layer with these short labels:
Scorecard
Friction Register
Validation Matrix
Decision Log

Make the visual imply speed, control, and repeatability. Include subtle arrows showing the loop repeats after adoption.

Do not include paragraphs, tiny dashboard text, fake company logos, robots, science fiction imagery, or uncontrolled automation. Make it board-deck ready with generous spacing and readable labels.
```

## Variant: More Executive

```text
Create a cleaner and more senior-executive version of the same image. Reduce visual complexity, use only five main labels, and make "Validated Adoption" feel like a controlled decision point rather than an automatic deployment.
```

## Variant: More Operational

```text
Create a slightly more operational version that shows participant branches feeding into a group review checkpoint before adoption. Keep the same 16:9 slide format and avoid dense text.
```

## Suggested Slide Title

```text
From AI Tool Usage To A Governed Engineering Improvement Loop
```
